### yarn - 前端包管理工具
yarn 命令
**初始化  yarn init -y
安装   yarn add [--dev|-D]【--prod】
全局安装 yarn global add '包'**


yarn 常用命令
**查看包信息  yarn info ‘包’
列举已经安装的包 yarn list
检查是否更新  yarn outadated  更新  yarn upgrade
移除 yarn remove**


yarn 特性
**yarn check 检查package.json 和 yarn.lock 是否一致
yarn audit 检查安装的包存在哪些漏洞
yarn create 脚手架**
